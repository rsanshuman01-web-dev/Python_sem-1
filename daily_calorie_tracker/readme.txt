Daily Calorie Tracker (CLI)

This project is a simple Python-based Command Line Interface (CLI) tool that helps users record their daily meals and calories. It calculates the total and average calorie intake and checks whether the user is within their daily calorie limit.

Features

Input multiple meal names and calorie values

Calculate total and average calorie intake

Daily limit warning system

Neatly formatted summary output

Option to save the session log to a text file

How It Works

The program asks how many meals the user wants to enter.

For each meal, the user inputs the meal name and calorie amount.

The program calculates:

Total calories

Average calories

The user enters their daily calorie limit.

The program shows:

A formatted summary table

A warning if the limit is exceeded

The user can choose to save the summary to a file (calorie_log.txt).

Technologies Used

Python

Basic I/O

Lists

Loops

File Handling

How to Run

Install Python 3

Save the script as calorie_tracker.py

Open terminal/command prompt

Run:

python calorie_tracker.py
